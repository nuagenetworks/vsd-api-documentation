    Attributes documentation

    allowAdvancedQOSConfiguration boolean
    Discussion

    Controls whether this enterprise has access to advanced QoS settings.

    Charateristics

    Format: free
    SDK attribute: allowAdvancedQOSConfiguration

    allowedForwardingClasses list
    Discussion

    Allowed Forwarding Classes for this enterprise. Possible values are NONE, A, B, C, D, E, F, G, H, .

    Charateristics

    Allowed values: A, B, C, D, E, F, G, H, NONE
    Format: free
    SDK attribute: allowedForwardingClasses

    allowGatewayManagement boolean
    Discussion

    If set to true lets the enterprise admin create gateway templates and instances.

    Charateristics

    Format: free
    SDK attribute: allowGatewayManagement

    allowTrustedForwardingClass boolean
    Discussion

    Controls whether QoS policies and templates created under this enterprise set the trusted flag to true

    Charateristics

    Format: free
    SDK attribute: allowTrustedForwardingClass

    BGPEnabled boolean
    Discussion

    Enable BGP for this enterprise profile

    Charateristics

    Format: free
    SDK attribute: BGPEnabled

    description string filterable  
    Discussion

    A description of the enterprise/organisation profile.

    Charateristics

    Format: free
    Min lenght: 1
    Max lenght: 255
    SDK attribute: description

    DHCPLeaseInterval integer
    Discussion

    DHCP Lease Interval (in hours) to be used by an enterprise.

    Charateristics

    Format: free
    Min value: 12
    Max value: 60
    SDK attribute: DHCPLeaseInterval

    DPIEnabled boolean filterable   orderable  
    Discussion

    Enable DPI for this Enterprise Profile

    Charateristics

    Default value: false
    SDK attribute: DPIEnabled

    enableApplicationPerformanceManagement boolean filterable   orderable  
    Discussion

    Enable DPI for this enterprise

    Charateristics

    Default value: false
    SDK attribute: enableApplicationPerformanceManagement

    encryptionManagementMode enum
    Discussion

    encryption management mode for this enterprise Possible values are DISABLED, MANAGED, .

    Charateristics

    Allowed values: DISABLED, MANAGED
    Format: free
    SDK attribute: encryptionManagementMode

    entityScope enum autogenerated   filterable   orderable  
    Discussion

    Specify if scope of entity is Data center or Enterprise level

    Charateristics

    Allowed values: ENTERPRISE, GLOBAL
    Format: free
    SDK attribute: entityScope

    externalID string filterable   orderable   locally unique  
    Discussion

    External object ID. Used for integration with third party systems

    Charateristics

    Format: free
    SDK attribute: externalID

    floatingIPsQuota integer
    Discussion

    Quota set for the number of floating IPs to be used by an enterprise.

    Charateristics

    Max value: 250000
    SDK attribute: floatingIPsQuota

    lastUpdatedBy string autogenerated  
    Discussion

    ID of the user who last updated the object.

    Charateristics

    Format: free
    SDK attribute: lastUpdatedBy

    name string required   filterable   orderable  
    Discussion

    The unique name of the enterprise. Valid characters are alphabets, numbers, space and hyphen( - ).

    Charateristics

    Format: free
    Min lenght: 1
    Max lenght: 255
    SDK attribute: name

    receiveMultiCastListID string
    Discussion

    Charateristics

    Format: free
    SDK attribute: receiveMultiCastListID

    Discussion

    Charateristics

    Format: free
    SDK attribute: sendMultiCastListID
